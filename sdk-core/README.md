# SAP CDC (Gigya) Android SDK

## Description
Android core SDK.

## Requirements
The core Android SDK requires SDK 14 and above.
For Biometric capabilities, SDK 23 and above is required.

## Download and Installation
* Clone the repo.
* Build the project using Gradle.
